template<typename T , IvyPointerType IPT>
__HOST__ bool std_ivy::IvyUnifiedPtr< T, IPT >::transfer ( IvyMemoryType const &  new_mem_type,
bool  transfer_all,
bool  release_old

Transfer pointer storage and metadata to a new memory domain.

transfer: Transfers the memory type of the pointer to the new memory type. If transfer_all is true, pointers ref_count_ and mem_type_ are also transferred. Otherwise, these two pointers are created in the default memory location of the execution space.

See also transfer_impl for the internal implementation, allowing the user to also make a new copy of the pointer.

Parameters
new_mem_typeDestination memory domain.
transfer_allWhether metadata pointers also move to destination domain.
release_oldWhether to release old allocations after transfer.
Returns
True on success.

transfer_impl: Implementation for transferring the memory type of the pointer to the new memory type. If transfer_all is true, pointers ref_count_ and mem_type_ are also transferred. Otherwise, these two pointers are created in the default memory location of the execution space. If copy_ptr is true, a new pointer is created. If release_old is true, the old pointers are released.
