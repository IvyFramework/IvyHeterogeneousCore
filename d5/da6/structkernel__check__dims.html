Detailed Description

template<typename Kernel_t>
struct kernel_check_dims< Kernel_t >

Dimension-check policy for kernel dispatch.

Users may specialize this policy for custom kernels to tailor boundary checks used by run_kernel and generic_kernel_[N]D launch wrappers.

Default behavior checks whether thread indices are within input bounds (e.g. i<n). On GPU, this protects against out-of-range threads caused by launch-grid rounding.

Example usage in a kernel implementation: if (kernel_check_dims<MyKernel>::check_dims(i, n)) { ... }.
