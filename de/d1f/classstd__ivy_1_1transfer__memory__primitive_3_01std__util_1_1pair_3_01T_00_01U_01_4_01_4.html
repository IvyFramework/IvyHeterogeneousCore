Detailed Description

template<typename T, typename U>
class std_ivy::transfer_memory_primitive< std_util::pair< T, U > >

transfer_memory_primitive<std_util::pair<T, U>>: std_util::pair is a container with internal memory, so we can use this object safely only when internal memory transfer is ensured. This specialization ensures that internal memory transfer routines are called for the first and second elements of the pair.
